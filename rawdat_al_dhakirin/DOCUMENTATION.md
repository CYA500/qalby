## الخطوات التالية للتشغيل

### 1. إنشاء مشروع Flutter جديد (إذا لزم الأمر)
```bash
flutter create rawdat_al_dhakirin
cd rawdat_al_dhakirin
```

### 2. استبدال الملفات
انسخ جميع الملفات المنشأة إلى المجلد المناسب:
- انسخ `lib/` إلى مجلد المشروع
- انسخ `pubspec.yaml` إلى جذر المشروع
- انسخ `README.md`

### 3. تثبيت الحزم
```bash
flutter pub get
```

### 4. تشغيل التطبيق
```bash
# على المحاكي أو الجهاز
flutter run

# أو بناء APK
flutter build apk --release
```

## مميزات كل شاشة

### 1. شاشة المسبحة (TasbeehScreen)
- زر دائري كبير نابض عند الضغط
- 3 عدادات: الجلسة / اليوم / الإجمالي
- حلقة تقدم دائرية (CircularProgressIndicator)
- اهتزاز عند كل ضغطة (Vibration)
- تبديل بين 10 أذكار بالأسهم
- أهداف قابلة للتغيير (33/100/500)
- رسالة تهنئة عند اكتمال الهدف
- حفظ تلقائي في localStorage

### 2. شاشة الأحاديث (HadithScreen)
- حديث اليوم في البanner العلوي
- شبكة فئات (GridView) مع أيقونات
- 7 فئات: الإيمان، الصلاة، الذكر، الأخلاق، الحياة اليومية، المغفرة، الجنة
- بطاقات قابلة للتوسيع (Expansion)
- عرض الراوي والمصدر والشرح
- عداد الأحاديث لكل فئة

### 3. شاشة الآيات (VersesScreen)
- 10 آيات منوعة
- فلترة حسب 6 فئات: الصبر، التقوى، التوحيد، الدعاء، التوكل، الطمأنينة
- عرض الآية بخط Scheherazade (خط مصحف)
- تفسير مبسط لكل آية
- درس مستفاد
- بطاقات ثلاثية الطبقات (آية → تفسير → درس)

### 4. شاشة الأذكار (AdhkarScreen)
- 3 تبويبات: صباح / مساء / نوم
- أذكار الصباح: 5 أذكار
- أذكار المساء: 4 أذكار
- أذكار النوم: 4 أذكار
- عداد تكرار لكل ذكر
- إشارة عند الإكمال (اللون الأخضر)
- مصدر الحديث لكل ذكر

## التخزين المحلي (SharedPreferences)

المفاتيح المستخدمة:
- `tasbeeh_session`: عداد الجلسة الحالية
- `tasbeeh_daily`: عداد اليوم (يتصفر تلقائياً كل يوم جديد)
- `tasbeeh_grand`: الإجمالي الكلي
- `current_dhikr`: مؤشر الذكر الحالي
- `is_dark_mode`: حالة الوضع المظلم
- `daily_hadith`: مؤشر حديث اليوم
- `last_day`: آخر يوم تم فيه الحفظ (للفحص اليومي)
