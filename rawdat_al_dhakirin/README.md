# روضة الذاكرين | Rawdat Al-Dhakirin

تطبيق إسلامي للإنتاجية الروحية - Islamic Productivity App

## 📱 المميزات الرئيسية

### 1. المسبحة الإلكترونية (Digital Tasbeeh)
- ✅ 10 أذكار مبرمجة (سبحان الله، الحمد لله، الله أكبر، لا إله إلا الله، وأكثر)
- ✅ 3 أنظمة عداد (الجلسة - اليوم - الإجمالي الكلي)
- ✅ حلقة تقدم دائرية متحركة
- ✅ اهتزاز عند الضغط (haptic feedback)
- ✅ تغيير الأذكار بالسحب
- ✅ بطاقة فضل الذكر لكل ذكر

### 2. مكتبة الأحاديث (Hadith Library)
- ✅ 7 فئات: الإيمان، الصلاة، الذكر، الأخلاق، الحياة اليومية، المغفرة، الجنة
- ✅ 27+ حديث نبوي شريف
- ✅ حديث اليوم (يتغير تلقائياً)
- ✅ بطاقات قابلة للتوسيع مع الشرح والراوي

### 3. آيات سهلة الحفظ (Quranic Verses)
- ✅ 10 آيات من القرآن الكريم
- ✅ 6 تصنيفات: الصبر، التقوى، التوحيد، الدعاء، التوكل، الطمأنينة
- ✅ تفسير مبسط لكل آية
- ✅ درس مستفاد من كل آية
- ✅ فلترة حسب التصنيف

### 4. الأذكار اليومية (Daily Adhkar)
- ✅ أذكار الصباح (5 أذكار)
- ✅ أذكار المساء (4 أذكار)
- ✅ أذكار النوم (4 أذكار)
- ✅ عداد تكرار لكل ذكر
- ✅ مصدر الحديث لكل ذكر

## 🎨 التصميم

- **اللون الرئيسي**: زيتوني (#5D7B6F) - يرمز للهدوء الروحي
- **اللون الثانوي**: ذهبي (#D4A373) - يرمز للأصالة الإسلامية
- **الخطوط**: Amiri، Scheherazade New (خطوط عربية أصيلة)
- **الاتجاه**: RTL (كامل الدعم للعربية)
- **الوضع المظلم**: دعم كامل للوضع المظلم مع حفظ التفضيل

## 🛠️ التقنيات المستخدمة

- **Flutter 3.x**: إطار العمل الرئيسي
- **Shared Preferences**: التخزين المحلي للعدادات والإعدادات
- **Google Fonts**: الخطوط العربية
- **Vibration**: الاهتزاز عند التسبيح
- **Material Design 3**: التصميم الحديث

## 🚀 طريقة التشغيل

### 1. متطلبات النظام
- Flutter SDK 3.0.0 أو أحدث
- Dart SDK 3.0.0 أو أحدث
- Android SDK (للأندرويد)
- Xcode (للي iOS)

### 2. تثبيت الحزم
```bash
flutter pub get
```

### 3. تشغيل التطبيق
```bash
# للوضع التطويري
flutter run

# بناء APK للأندرويد
flutter build apk --release

# بناء لـ iOS
flutter build ios --release
```

## ⚙️ المميزات التقنية

### إدارة الحالة
- استخدام `setState` لإدارة الحالة المحلية
- التخزين المستمر باستخدام `SharedPreferences`
- إعادة تعيين يومية تلقائية للعداد اليومي

### الأداء
- استخدام `IndexedStack` للحفاظ على حالة الشاشات
- تحميل البيانات مرة واحدة عند البدء
- رسوم متحركة محسّنة باستخدام `AnimationController`

### تجربة المستخدم
- دعم كامل للغة العربية (RTL)
- اهتزاز عند الضغط على المسبحة
- رسوم متحركة سلسة
- تصميم متجاوب مع مختلف أحجام الشاشات
